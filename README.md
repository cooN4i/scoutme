# Scoutme - платформа футбольных талантов
Сервис, на котором пользователи могут создавать анкеты для привлечения футбольных скаутов

## Фичи
* **Безопасный вход по JWT-токенам**
* **Разделение ролей**: обычный пользователь и администратор
* **Права доступа**: каждый пользователь управляет только своей анкетой, кроме админстратора - у него есть доступ к каждой из анкет
* **Фильтрация и пагинация анкет**

## Стек
* FastAPI
* SQLAlchemy
* Pydantic
* aiosqlite

## Запуск проекта
1. Создание виртуального окружения
```bash
python -m venv venv
```

2. Установка зависимостей
```bash
pip install -r requirements.txt
```

3. Запуск сервера
```bash
uvicorn main:app --reload
```

4. Сайт откроется по адресу http://127.0.0.1:8000, документация доступна по адресу http://127.0.0.1:8000/docs
